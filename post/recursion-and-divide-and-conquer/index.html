Recursion & Divide and Conquer

Author: Lesenelir's Blog Date: 2020-07-09 Reading Time:1.5 mins words:337 Category: Data Structure and Algorithm

1.递归与分治

概述

分治法：通过求解局部性的小问题来解开原本的问题。
分治步骤：

  1. 将问题分割成局部问题
  2. 递归求解局部问题
  3. 将局部问题进行整合，解决原问题

2.穷举搜索

概述

对于穷举搜索来说，每一个元素都有选和不选两种选择

线性搜索问题描述

代码实现

#include <cstdio>
#include <iostream>
using namespace std;


int n;
int A[50];

int solve(int i , int m) {
	if (m == 0) return 1;
	if (i >- n) return 0;
	int res = solve(i + 1, m) || solve(i + 1, m - A[i]);
	return res;
}


int main() {
	int q, M;
	scanf("%d", &n);
	for (int i = 0; i < n; i++) {
		scanf("%d", &A[i]);
	}
	scanf("%d", &q);
	for (int i = 0; i < q; i++) {
		scanf("%d", &M);
		if (solve(0, M) == 1) printf("YES\n");
		else printf("NO\n");
	}	
	return 0;
}

算法解析

solve(int i, int m) 表示用第i个元素后面的元素能得到m时返回true。
这样做的好处是，能够把solve(i, m)分解成solve(i+1, m)和solve(i+1, m-A[i])， m-A[i]表示使用了第i个元素。
最后递归的原问题是solve(0, m)。

递归会多次进行计算solve(i, m)，这样会有很多重复计算，我们通过动态规划可以解决这个问题。
